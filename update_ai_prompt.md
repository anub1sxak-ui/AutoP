# Обновление промпта AI Agent

Текущий промпт AI Agent ожидает текст из PDF, но теперь мы передаем base64 данные.

## Варианты решения:

1. **Добавить HTTP Request узел** для извлечения текста из PDF через API (pdf.co, Adobe PDF Services и т.д.)
2. **Обновить промпт AI**, чтобы он мог работать с base64 PDF напрямую (если AI поддерживает)

## Рекомендация:

Лучше добавить HTTP Request узел между "Extract PDF Text" и "AI Agent" для извлечения текста из PDF через внешний API, так как это более надежно.

Но если AI Agent (DeepSeek) поддерживает работу с base64 PDF, можно обновить промпт.
