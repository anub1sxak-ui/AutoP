# Инструкция по импорту Workflow

## Способ 1: Импорт через веб-интерфейс (РЕКОМЕНДУЕТСЯ)

1. Откройте https://n8n.anub1sx.work
2. Войдите в систему
3. В правом верхнем углу нажмите на меню (три точки) или кнопку "+"
4. Выберите **"Import from File"** или **"Import"**
5. Выберите файл `workflow_pdf_lenses_price_analyzer.json`
6. Workflow будет импортирован и откроется в редакторе

## Способ 2: Через API (если есть API ключ)

Если у вас есть API ключ n8n, можно импортировать через команду:

```bash
curl -X POST \
  -H "X-N8N-API-KEY: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  https://n8n.anub1sx.work/api/v1/workflows \
  -d @workflow_pdf_lenses_price_analyzer.json
```

## После импорта

1. **Настройте credentials:**
   - Telegram Bot API (для узла Telegram Trigger)
   - DeepSeek API (для узла AI Agent)
   - Google Sheets API (для узла Write to Google Sheets)

2. **Настройте Google Sheets:**
   - Создайте таблицу с колонками (см. список ниже)
   - В узле "Write to Google Sheets" укажите ID вашей таблицы

3. **Активируйте workflow:**
   - Переключите тумблер "Active" в правом верхнем углу
